# Proyecto_Final
#### Barun Maharjan

___

Dataset utilizado: https://github.com/rfordatascience/tidytuesday/tree/main/data/2022/2022-09-13

El dataset señala todos los casos reportados de los avistamientos de Bigfoot (Pie Grande) en los Estados Unidos desde los años 1900.

___ 

El dataset proporcionaba 28 variables.

Los variables en que decidí enfocar fueron: **county, state, date, latitude & longitude**

___

El documento .Rmd es el registro del proceso, el documento hecho en .qmd es el sitio final

___
