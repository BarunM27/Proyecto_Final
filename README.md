# Proyecto_Final

___

Dataset utilizado: https://github.com/rfordatascience/tidytuesday/tree/main/data/2022/2022-09-13

El dataset señala todos los casos observados de los avistamientos de Bigfoot en los Estados Unidos desde los años 1900
___ 

El dataset proporcionaba 28 variables.

Los variables en que decidí enfocar fueron: **county, state, date, latitude & longitude**

## Preguntas para contestar

¿Qué estados tienen la mayor cantidad de avistamientos?

¿Qué condado del estado de Washington tiene la mayor cantidad de avistamientos?

¿Qué temporada tiene la mayor cantidad de avistamientos?

¿En qué años fueron más prevalentes los avistamientos?

¿Cuál es el patrón?
